# Trainings Tracker

Lokale Mini-App zum Eintragen und Speichern von Trainings.

## Starten

- Doppelklick auf [Trainings Tracker.app](/Users/TUM/Desktop/USC%20Minijob%20Tracker/Trainings%20Tracker.app)
- Falls die App von macOS blockiert wird: Rechtsklick auf `Trainings Tracker.app` und `Oeffnen`
- Alternative: [start.command](/Users/TUM/Desktop/USC%20Minijob%20Tracker/start.command) ausfuehren

## Stoppen

- Doppelklick auf [Trainings Tracker Stop.app](/Users/TUM/Desktop/USC%20Minijob%20Tracker/Trainings%20Tracker%20Stop.app)
- Alternative: [stop.command](/Users/TUM/Desktop/USC%20Minijob%20Tracker/stop.command) ausfuehren

## Daten sichern

- Alle Daten liegen in [data.json](/Users/TUM/Desktop/USC%20Minijob%20Tracker/data.json)
- Diese Datei regelmaessig kopieren, wenn du ein Backup willst
- Beim Wiederherstellen einfach die bestehende `data.json` durch dein Backup ersetzen
